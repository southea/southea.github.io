实验目的：

熟悉使用主流云平台并搭建个人博客或者网站。

具体包括：

参考方案：基于主流云平台，设计、实现个人博客或者网站的搭建，撰写实验报告（附带网站链接并可以访问），并在网站上呈现此次实验报告。

硬件环境：

联网的计算机一台

软件环境：

Windows or Linux

实验步骤与内容：

环境准备

安装node.js和git

在github账号上创建仓库

配置安装hexo

用淘宝镜像源安装一直失败，可能是证书失效的问题，切换成别的镜像源成功：

npm config set registry https://registry.npmjs.org/

设置ssh：

上传博客

安装上传工具：npm install hexo-deployer-git

新建文章：hexo new “name”

预览 hexo s

生成文件 hexo g

部署到github hexo d

打开页面Page not found · GitHub Pages (southea.github.io)发现404，是因为没有部署成功，重新部署：发现是没有安装一个依赖，重新安装：npm install hexo-deployer-git –save

上传文件：

博客地址：Hexo (southea.github.io)

结论分析与体会

​ 这次实验让我对个人博客的搭建有了深入的了解。通过这次实验，我接触到了技术工具如git、hexo、markdown等等，熟悉了这些工具的使用，提升了自己的技术能力。同时了解到个人博客是一个分享知识、经验和见解的平台。通过写作和分享，可以将自己学到的知识整理成文章，与他人分享，并从中得到反馈和讨论。这有助于巩固自己的学习成果，同时也可以帮助他人解决问题，建立起良好的学习和交流氛围。在之后的学习生活中，我也会尝试写博客记录分享自己的成长。
